<h1>Simple Calculator</h1>
<p>This is a simple calculator application built using Kotlin and Jetpack Compose. The application allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.</p>
<h2>Getting Started</h2>
<p>To get started with this project, you will need to have a basic understanding of Kotlin and Jetpack Compose, as well as experience with building mobile apps.</p>
<h2>Prerequisites</h2>
<ul>
    <li>Android Studio</li>
    <li>Kotlin</li>
    <li>Jetpack Compose</li>
</ul>

<h2>Installing</h2>
<ol>
    <li>Clone the repository to your local machine using the following command:
<pre>
git clone https://github.com/<el-sayed-mustafa>/SimpleCalculator.git
</pre>
    </li>
    <li>Open the project in Android Studio</li>
    <li>Build and run the application on an emulator or a physical device</li>
</ol>
